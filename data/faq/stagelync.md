# StageLync import

1. Open the **StageLync** tab.
2. Click **Sync fixture** to load the local StageLync people fixture.
3. Use **Discover** with a query (for example `elite aerial UAE`).
4. Click **Import to OLC** to create a linked talent (`TAL-SL-*`).
5. Run a match; StageLync export prefers linked IDs when present.

Live StageLync OAuth is not enabled in this POC — sync uses a local fixture only.
